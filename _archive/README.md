# Archive

Previous MatchYard designs and site files, kept for reference. **Not deployed**
(excluded via `.vercelignore`). Archived July 2026 when the current site
(dark editorial "Switzer" design) became the finalized production version.

## Contents

### `old-site/`
The original MatchYard landing site (shared `style.css` design system):
- `old-landing-index.html` — the previous homepage
- `blog.html`, `privacy.html`, `terms.html` — old supporting pages
- `style.css` — the old shared stylesheet
- `og-image.svg`, `logo1.png`, `badge-*.svg`, `shots/`, `uploads/` — old assets

### `theme-variants/`
Exploratory landing-page theme concepts that preceded the final design:
- `theme-navy/` — midnight navy + volt lime
- `theme-green/` — volt lime + teal
- `theme-velocity/` — navy + blue + orange
- `theme-orbit/` — cosmic purple + coral
- `theme-club/` — light editorial cream
- `theme-widget.html` — theme comparison widget

### `docs/`
- `MatchYard-Technology-Cost-Plan.pdf` — infrastructure cost analysis

The live production site lives at the repository root (`index.html`, `support.js`,
`blog/`, `assets/`).
